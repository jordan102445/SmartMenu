# Menu Images

Place real dish photos here when you want to replace the temporary Unsplash images.

Use web-friendly images:

- `.webp`, `.jpg`, or `.png`
- Around 1200px wide
- Compressed for mobile

Then point `src/data/menu.json` to the image:

```json
{
  "image": "/images/truffle-margherita.webp"
}
```
